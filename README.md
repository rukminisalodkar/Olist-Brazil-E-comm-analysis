# Olist-Brazil-E-comm-analysis
End-to-end data analysis project on sales data of Olist Stores in Brazil using SQL for exploratory data analysis and Power BI for visualization. 
The initial analysis (EDA) was conducted through SQL queries by establishing a connection in Jupyter notebook and Power BI and Power Query has been used to effectively visualise the data and present in the format of a dashboard to give recommendations
on the direction of the business.
